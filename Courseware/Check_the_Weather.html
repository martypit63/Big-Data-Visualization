Check the Weather

Wednesday, September 13, 2017 1:30 PM

Exercise 2: Check the weather

Duration: 20 mins

Synopsis: In this exercise, attendees will integrate the 10-day weather forecast into the sample solution.

Task 1: Register for trial account in WeatherUnderground.com

  1. To retrieve the 10-day hourly weather forecast, you will use an API from WeatherUnderground.com. There is a free developer version that provides you access to the API you need for this workshop.

  2. Navigate to http://www.wunderground.com/weather/api/.

  3. Click Explore My Options.

  1. On the Get Your API Key page, select Anvil Plan.
  1. Scroll down until you see the area titled How much will you use our service? Ensure Developer is selected.
  1. Click Purchase Key.
  1. Complete the Create an Account form by providing your email, username and a password and agreeing to the terms. Click Basic â€“ Free.

  1. Back on the Get Your API Key page, re-select Anvil and click Purchase Key.

  2. In a few moments you should receive a confirmation email at the email address you provided. Click the link found within the email.

  1. On the page that loads, you will see a message like the following:
  1. Complete the brief contact form. When answering where will the API be used, select Website. For Will the API be used for commercial use, select No. Click Purchase Key.
  1. You should be taken to a page that displays your key, similar to the following:

  1. Take note of your API Key. It is available from the text box labeled Key ID.

  2. To verify that your API Key is working, modify the following URL to include your API Key: http://api.wunderground.com/api/<YOURAPIKEY>/hourly10day/q/CA/SEATAC.json.

  3. Open your modified link in a browser, you should get a JSON result showing the 10-day, hourly weather forecast for the Seattle-Tacoma International Airport.

Task 2: Update sample Web App with API Key

  1. Next, you will update the sample app with your Weather Underground API Key.

  2. With the sample app open in Visual Studio, in Solution Explorer, right-click the project and select Properties.

  3. Click the Settings tab.

  1. Edit the value for the Weather_APIKey and click Save.

  2. In Visual Studio Solution Explorer, right-click your project, select Publish, and then click Publish.

  3. On your deployed web app, change the date of the flight to tomorrow's date (or another future date no further than 10 days away).

  4. Click Predict Delay. You should see the weather icon display. If you hover over the icon, you will get the description of the weather conditions forecasted. At this point, Delay Prediction should display not configured. You will set Delay Prediction up shortly.
